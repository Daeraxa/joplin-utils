# Plugins

Let's start with a short list of my note taking related plugins that implement a specific feature and can be combined.

  • Markdown All in One (opens new window): Markdown language support
  • markdown lint (opens new window): Markdown linter tool
  • Prettier (opens new window): multi-language formatting tool, very good support for markdown
  • PicGo (opens new window): image upload (this is a third-party program)
  • Markdown PDF (opens new window): Markdown export PDF
  • Draw.io Integration (opens new window): draw.io Flowchart Integration
    • Markdown Preview Mermaid Support (opens new window): mermaid flowchart support DSL, but draw.io is better
  • vscode-mindmap (opens new window): mind map integration (this is the fork version, but it's much better)
  • Local History (opens new window): local history
  • Code Spell Checker (opens new window): Word spell checker
  • joplin-vscode-plugin (opens new window): joplin notes integration

You may notice that some of these plugins have nothing to do with markdown, but they are still important for editing, for example the spell checker feature that was only recently implemented in Joplin (v1.4.*) already exists in the existing VSCode plugins.
